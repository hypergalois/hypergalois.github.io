# Hypergalois

Research engineering portfolio built with Astro 6, Tailwind CSS v4, MDX content
collections, static GitHub Pages deployment, and a production-grade component
foundation.

## Commands

```bash
pnpm install
pnpm dev
pnpm run build
pnpm run check
```

## Deployment

The site deploys to GitHub Pages from `.github/workflows/deploy.yml`.

Production URL:

```text
https://hypergalois.github.io
```
