## Problem

Post-quantum certificates stress infrastructure that was tuned for smaller keys,
older assumptions, and narrower interoperability profiles.

## Technical record

The project treats X.509 as an operational system: parse the certificate, inspect
the chain, test the boundary, and record where software expectations diverge from
new cryptographic material.

## Evidence

The assurance layer records parser behavior, validator decisions, and regression
cases under PQC certificate structures.

## Stack

- Certificate parsing and linting
- Chain validation experiments
- PQC algorithm metadata
- Regression-oriented test cases
