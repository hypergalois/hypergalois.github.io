## Problem

Post-quantum migration changes deployed behavior across handshake size,
certificate chains, middleboxes, fallback paths, and operational defaults.

## Artifact

This project is framed as an observatory: a repeatable measurement pipeline that
collects protocol evidence, normalizes it, and turns deployment behavior into
inspectable data.

## Evidence

The pipeline links migration claims to traces, normalized outputs, and static
reports that can be compared across runs.

## Stack

- TLS probing and packet capture
- Certificate-chain inspection
- Reproducible experiment runners
- Static reporting for long-lived evidence
