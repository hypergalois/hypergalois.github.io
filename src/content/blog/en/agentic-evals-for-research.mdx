Agentic systems produce fluent runs with uneven reliability. Evaluation needs an
inspectable trace.

The evaluation object includes the task, tools, intermediate decisions, files
touched, failures, recovery behavior, and final evidence. That record supports
comparison across agents, prompts, and tool environments.

## A better harness

- Scenario inputs
- Tool-call trace
- File diffs and outputs
- Failure labels
- Human review records

The harness turns an agent run into a reviewable object: inputs, actions,
outputs, failures, and final state.
