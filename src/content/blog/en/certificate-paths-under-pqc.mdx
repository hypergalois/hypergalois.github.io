X.509 is a standards ecosystem, a parser ecosystem, and an operational ecosystem.
Post-quantum migration touches all three.

Large keys and signatures change byte size, parser stress, path-building behavior,
transport limits, and tooling paths that rarely exercise those boundaries.

## Questions worth asking

- Which fields and extensions change shape?
- Which validators reject unusual but valid structures?
- Where do chain sizes become operationally relevant?
- What does a useful lint report need to preserve?

Assurance work ties compatibility claims to validators, corpus cases, and
reproducible lint reports.
